h1. Emailbolt

This application creates disposable private emails. If your email address is ever compromised you can destroy that address and remove any link to your main address.

h2. Ruby on Rails

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.1

Learn more about "Installing Rails":http://railsapps.github.io/installing-rails.html.

h2. Database

This application uses SQLite with ActiveRecord.

h2. Development

* Template Engine: ERB
* Testing Framework: Test::Unit
* Front-end Framework: Zurb Foundation 5
* Form Builder: SimpleForm
* Authentication: OmniAuth
* Authorization: None
